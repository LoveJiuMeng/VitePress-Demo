## KMS激活Windows

1. 以管理员方式打开Passwall

2. 输入命令安装密钥 


```
slmgr /ipk W269N-WFGWX-YVC9B-4J6C9-T83GX
```

2. 设置KMS服务器

```
slmgr /skms Kms服务器地址
slmgr /skms 172.22.2.8
```

3. 输入激活命令

```bat
slmgr /ato
```
